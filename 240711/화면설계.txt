메인화면
1 로고 클릭 시 main(메인 화면)로 이동
2. 작업실 클릭 시 workspace(작업실 메인 화면)로 이동
3. 라이브러리 클릭 시 library(라이브러리 메인 화면)로 이동
4. 커뮤니티 클릭 시 community(커뮤니티 메인 화면)로 이동
5. 마켓 클릭 시 market(마켓 메인 화면)로 이동
6. 로그인 클릭 시 userLogin(사용자 로그인 화면)로 이동
7. 회원가입 클릭 시 join(회원가입 화면)로 이동


메인화면2
8. 메인 캐러셀 클릭 시 연관 페이지로 이동, 화살표 클릭 시 다음 이미지로 이동
9. 인기게시글에 표시된 5개의 목록 중 하나를 클릭하면 해당 게시글 화면으로 이동하고, 더보기 클릭 시 community(커뮤니티 메인 화면)의 인기게시글 탭으로 이동
10. 작업실에 표시된 5개의 목록 중 하나를 클릭하면 해당 작업물 화면으로 이동하고, 더보기 클릭 시 workspace(작업실 메인 화면)로 이동
11. 공연 포스터 클릭 시 해당 공연 상세 정보 페이지로 이동하고, 더보기 클릭 시 show(공연 정보 메인 화면)로 이동


메인화면3
12. 사진 썸네일 클릭 시 해당 상품 상세 정보 페이지로 이동하고, 더보기 클릭 시 market(마켓 메인 화면)로 이동
13. 광고 캐러셀 이미지 클릭 시 연관 페이지로 이동, 화살표 클릭 시 다음 이미지로 이동


푸터
14. 공지사항 클릭 시 notice(공지사항 메인 화면)로 이동
15. 운영참여 클릭 시 operation(운영참여 메인 화면)로 이동
16. 개인정보처리방침 클릭 시 privacyPolicy(개인정보처리방침 페이지)로 이동
17. 이용약관 클릭 시 termsOfUse(이용약관 페이지)로 이동



로그인 화면
1. 아이디와 비밀번호 입력
2. 로그인 버튼 클릭 시 유효하지 않은 아이디나 잘못된 비밀번호 등 오류 발생 시 알림창 출력 후 다시 로그인 화면 출력. 이 때 기존에 입력한 아이디 및 비밀번호 값이 그대로 유지됨
3. 인스타 로그인 클릭 시 인스타그램 로그인 화면으로 이동
4. 회원가입 클릭 시 join(회원가입 화면)로 이동
5. 아이디 / 비밀번호 찾기에서 아이디, 비밀번호 문자 클릭 시 각각 findId(아이디 찾기 페이지), findPwd(비밀번호 찾기 페이지)로 이동


회원가입 화면
1. 회원 가입에 필요한 정보들을 각각 입력
2. 중복확인 버튼 클릭 시 user 테이블의 id를 조회해 동일한 아이디가 있는지 확인 후 사용 가능 여부 팝업으로 출력
3. 가입하기 버튼 클릭 시 유효성검사 실행 후 올바르게 작성되었다면 joinFin(회원가입 완료 페이지)로 이동.
4. 취소버튼 클릭 시 main(메인 화면)로 이동


아이디 찾기
1. 아이디 찾기에 필요한 정보들을 각각 입력
2. 인증번호 받기 버튼 클릭 시 입력된 메일 주소가 user 테이블에 있는지 확인 후, 존재하면 메일 발송. 존재하지 않으면 알림창으로 안내
3. 다음 버튼을 눌렀을 때 발송된 인증번호와 입력한 인증번호가 일치하는지 유효성 검사 후 일치한다면 아이디 정보 관련 페이지로 이동


아이디 정보 확인
1. 아이디가 user 테이블에 존재할 경우 해당 아이디 출력
2. 로그인 버튼 클릭 시 userLogin(사용자 로그인 화면)로 이동
3. 비밀번호 찾기 클릭 시 findPwd(비밀번호 찾기 페이지)로 이동


아이디 정보 확인
1. 아이디가 user 테이블에 존재하지 않는 경우 아이디가 존재하지 않는다는 안내 문구 출력
2. 회원가입 버튼 클릭 시 join(회원가입 화면)으로 이동
3. 인스타회원가입 버튼 클릭 시 연동된 인스타그램 로그인 / 가입 화면으로 이동


비밀번호찾기
1. 비밀번호 찾기에 필요한 정보들을 각각 입력
2. 인증번호 받기 버튼 클릭 시 입력된 메일 주소가 user 테이블에 있는지 확인 후, 존재하면 메일 발송. 존재하지 않으면 알림창으로 안내
3. 다음 버튼을 눌렀을 때 발송된 인증번호와 입력한 인증번호가 일치하는지 유효성 검사 후 일치한다면 아이디 정보 관련 페이지로 이동


새 비밀번호 지정
1. 변경하고자 하는 비밀번호를 작성
2. 완료 버튼 클릭 시 새로 입력한 비밀번호 2개가 동일한지 확인 후, 일치한다면 비밀번호 변경 완료 알림창 안내 후 main(메인 화면)으로 이동. 일치하지 않는다면 알림창 출력 후 다시 비밀번호 입력 화면 출력. 이 때 기존에 입력한 값이 그대로 유지됨.


마켓 목록
1. 상단 카테고리 클릭 시 각각의 카테고리에 해당하는 게시글 목록 출력
2. 마켓 이용에 대한 공지사항 목록이 상단에 출력됨
3. 마켓 게시글이 10개씩 출력됨
4. 목록 페이지 이동
5. 검색 버튼 클릭 시 팝업 창 출력
7. 마켓 내에서만 검색하고 싶은 키워드 입력 후 검색 클릭 시 키워드가 포함된 글 목록 출력
6. 내가 쓴 글 버튼 클릭 시 기존에 본인이 작성했던 글 목록이 출력됨. 
7 글쓰기 버튼을 누르면 marketForm(마켓 글쓰기 화면)로 이동


마켓 글쓰기 화면
1. 제목 및 내용 각각 입력
2. 쓰고자 하는 글의 카테고리 선택
3. 웹 에디터를 이용하여 글 내용의 크기, 모양, 정렬 등을 변경
4. 사진 및 파일 첨부 버튼 클릭 시 사진과 파일을 추가할 수 있음.
5. 등록 게시판 클릭 시 제목과 내용에 내용이 모두 입력되어 있다면 글 등록 후 market(마켓 메인 화면)로 이동. 내용이 입력되어있지 않다면 알림창 출력


마켓 상세정보 화면
1. 추천 버튼 클릭 시 해당 게시물에 대한 추천 수가 1 증가
2. 목록 버튼을 누르면 market(마켓 메인 화면)로 이동.
3. 댓글 추천 버튼을 누르면 해당 댓글에 대한 추천 수가 1 증가
4. 댓글 입력 후 작성 버튼 클릭 시 해당 게시물에 대한 댓글이 업로드 됨. 내용이 입력되어있지 않다면 알림창 출력
